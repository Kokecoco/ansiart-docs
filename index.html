ansiart

ansiart is a Python library that converts images into text art using ANSI escape sequences, for output in Command Line Interfaces (CLI).

This tool also supports saving and loading of ANSI escape sequences as text files. By default, it uses True color; use the truecolor_to_256color function as needed for conversion.

Installation

pip install ansiart

Available Functions

  • image_to_ansi
  • save_ansi_art
  • load_ansi_art
  • truecolor_to_256color
